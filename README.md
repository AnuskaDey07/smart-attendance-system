# 🎓 Smart Attendance System

A Face Recognition Based Smart Attendance System developed using **Python, Flask, OpenCV, Face Recognition, SQLite, HTML, CSS, and JavaScript**. The system automates attendance management by identifying students through facial recognition and marking attendance in real time.

---

## 📌 Overview

Traditional attendance systems are time-consuming and prone to manual errors. This project provides an intelligent solution that uses computer vision and facial recognition technology to automatically identify students and record their attendance.

The system offers a user-friendly web interface for student enrollment, attendance tracking, and attendance management.

## 📖 How to Use

### Student Enrollment

1. Login to the system.
2. Open the Enrollment page.
3. Enter student details.
4. Start the camera.
5. Capture the student's face.
6. Save enrollment information.

### Attendance Marking

1. Open the Attendance page.
2. Start the face recognition scanner.
3. Stand in front of the webcam.
4. The system automatically identifies the student.
5. Attendance is marked and stored in the database.
